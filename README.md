# ICT Pulse 7

**Trends, Choices & Consequences** is a Grade 7 TLE-ICT digital learning experience aligned with the MATATAG Curriculum competency: **“Discuss the emerging trends and issues in ICT.”**

Prepared by **Althesa A. Sunio**, Teacher II (Junior High School), Estefania Montemayor National High School, for the MICT 502 Digital Curriculum Innovation Portfolio (Mid Year 2026).

## What is included

- All 12 portfolio sections required by the project guide
- Official MATATAG content standard and learning competency
- ADDIE development record and OBE alignment matrix
- Visible TPACK, SAMR, and UDL applications
- Google Workspace integration plan with instructional purposes
- Responsible AI prompt lab and five-step verification routine
- Diagnostic, formative, and performance assessments
- Instant feedback, answer key, remediation guide, and 16-point rubric
- Learner progress stored only in the local browser
- Responsive, keyboard-accessible, high-contrast, large-text, and print modes
- Optional five-minute presentation outline and script
- APA 7th-style reference list with clickable authoritative sources

## Files

- `index.html` — all portfolio and lesson content
- `styles.css` — visual design, responsive behavior, accessibility, and print layout
- `app.js` — quizzes, feedback, progress, display controls, prompt lab, and reflection tabs
- `.nojekyll` — tells GitHub Pages to serve the static files directly

No build tool or package installation is required.

## Publish with GitHub Pages

1. Create or open a public GitHub repository, such as `ict-pulse-7`.
2. Upload all four files to the repository root on the `main` branch.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`, then click **Save**.
6. Wait for GitHub to show the published address, usually:
   `https://YOUR-USERNAME.github.io/ict-pulse-7/`
7. Open that address in a private/incognito window and test the links, controls, quizzes, phone layout, and print mode.

## Privacy note

The built-in website quizzes do not send answers to a server. Completion data and display preferences use browser `localStorage` on the learner’s own device. The Reset link clears lesson progress. If Google Workspace is used during classroom implementation, use school accounts, appropriate sharing permissions, and school data-protection procedures.
